2015 臺大黑客松

此作品為臺大黑客松（2015-08-20 ~ 2015-08-23）比賽作品。

一直覺得要下載很多APP，才能知道路況、知道加油站在哪、知道哪裡有停車場嗎？首款結合加油站、停車場、施工、路況資訊

  1. 可以查看車流量
  2. 可以立即知道路上施工狀況
  3. 立即知道路況
  4. 查詢台北停車場、機車車位
  5. 可以查詢附近的加油站
  6. 查到什麼馬上地圖導引

台北的風，混雜著六百多萬人口，看似平靜的城市，其實充滿了未知的資訊，現在你只有打開裝置的 GPS 功能後，立刻查詢！

    *
    本文建置時間2015年06月左右，Rails是個進步很快的框架，如果過太久本文可能會失效，若您使用環境不同，可能會有不同的結果，但基本上如果是Yosemite版本都可以按照此流程。

首先，我以一個完全不懂的角度寫這篇文章，除了幫助自己記憶之外，希望能夠幫到跟我遇到同樣問題的人。然後，照理來說，如果你的電腦是個新的Yosemite，Mac應該會內建Ruby，不過之後我們還是會把他換成新的。

  • 安裝 Command Line Tools

有兩種方法，可自行選擇想要的。

  1. 直接至App Store 安裝最新版本的Xcode，按照安裝步驟安裝， 之前的教學可能會請你去安裝Commend Line Tools，請省略他，現已內建在Xcode中。
  2. 若覺得Xcode很占空間，自己又不寫iOS App的話，可以不安裝Xcode，選擇直接下載Command Line Tools，開啟Terminal後，輸入
    $ xcode-select --install

之後會跳出視窗，請按下

$ ruby -e “$(curl -fsSL https://raw.github.com/mxcl/homebrew/go/install)"

==>

這個指令幫你安裝RVM，並且在幫你安裝最新穩定版本的Ruby，上面跑完一堆東西安裝完了之後，重新啟動Terminal。
