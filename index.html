這個專案是跑在 ubuntu 加上 gunicorn，在使用 gunicorn 的時候，定時任務會重複啟動很多次，找了一下網路上的資源，問題在使用 gunicorn 對 flask 應用進行控制的時候如果設置了 gunicorn 的 --worker 參數大於 1 時，會出現一個定時任務執行多次的問題，此時要給 gunicorn 提供一個額外的 --preload 參數，這樣 flask 的 app 在 run 定時任務就只會執行一次。

『海軍陸戰隊軍事常備訓練役』後記

今天不聊技術，來聊聊當兵生活。當然了，請不要跟我戰我四個月很廢很怎樣，其實我也很不爽整個兵役制度，但是沒辦法，規定就是這樣，跟我戰也沒有實質意義。

那就切入正題了。是的，我消失的這四個月去服了兵役，最近要退伍了，趁著連假的時候來聊聊四個月兵役、海軍陸戰隊在幹嘛？

先介紹一下，我是4個月的海陸，全名是『海軍陸戰隊軍事常備訓練役』。

要帶的東西

首先，先幫各位整理要帶的東西：

  • 徵集令
  • 身分證
  • 健保卡
  • 手錶
  • 內褲
  • 錢（帶個兩千左右，然後盡可能換成零錢，進去你會知道零錢有多重要！）
  • 高中或大學的軍訓課折抵役期

各位應該會覺得，怎麼要帶的東西這麼少？是的，班長會帶各位去營長買，所以根本不用擔心。

第一次的交通

因為我的戶籍地在中壢，因此我是在中壢火車站報到。

當天會有區公所的人來帶各位，所以完全不用擔心。火車的部分，不用錢，坐莒光號，由於路途遙遠，車上會在中午供應免費的便當。

除了這台車會消失在火車時刻表上，窗外還會貼著『入營專車』，整台車死氣沈沈的，搞得很像什麼 地獄列車 之外，車上也是有賣飲料、零嘴。

到了左營火車站後，會有專車來載我們，整台車更是死氣沈沈，大概半小時時間，就會到龍泉新訓中心了。

一進去，就會到你所屬的連隊，順便告訴各位一個小口訣：快樂五、天堂六、地獄七、超級八。喔對，我就是在超級八的八連…所以我這宅宅都熬過去了，各位一定也可以。

新訓生活

一開始進軍營，會有班長來隨便抓人分班，好像都是分 1~9 班。分完班，知道自己哪排哪班（一定要記清楚）

前幾天真的很多事情，一進去就是先寫資料。會利用空餘時間帶大家去營站採買、拿裝備、換裝、剪頭髮（我這建議在外面先剪，不然髮婆一分鐘一顆頭，都用扯的）

第一個晚上洗澡很可怕，雖然不像以前那樣洗澡，衛浴設備也不錯，但是一堆人在擠，班長在後面喊，一直叫你快洗，所以也就索性三個人擠一間坦誠相見，但也不至於會要你撿肥皂。

然後第一個睡覺，真的很難熬，心理怕的要死，各種不適應，床不適應、枕頭不適應、沒有女友陪也不適應，真的超抖！

對了，現在都有冷氣了，所以倒是不用擔心太熱睡不著的問題。

接著是，第二天早上起床，起床後去連集合場集合，當然就是繼續罵一直罵不停罵，吃飯、繼續寫資料、調（幹）整（醮）你的裝備、互相認識同梯，前三天大概就都這樣過的。

在龍泉放過懇親後，時間就是用飛的，基本上就是照表操課…

首先一睜開眼就『早點名』，反正就跟著排隊，然後喊什麼『捍衛中華民國』『嚴守軍人武德』『一路陸戰隊』『終身陸戰隊』的精神口號

然後開始晨操，晨操就基本會有跑步、伏地挺身，其他的內容就看值星班長的心情

接著是單戰、手榴彈投擲、刺槍。

單戰就是要背誦一堆『抬頭觀察、由左至右、由近而遠…』『親愛的共軍，我們的檳榔跟你們的椰子一樣大』這種奇怪的台詞，然後還要你實際做出來，整個很羞恥之外，還要吃土、爬阿爬 滾啊滾、整個迷彩濕到內褲、內褲又濕回頭盔。

手榴彈投擲的部分，因為我手有痼疾，所以有『免計測證明』，如果你有痼疾，那也會建議去開一張，這張護身符真的很有用。

刺槍的話，就是拿著一個槍，在那刺來刺去，三公斤雖然看似沒什麼，拿久就很有什麼了（燦笑

還有我最喜歡的莒光課，就是寫寫大兵、看看莒光園地， 然後房業涵好正

喔對，有公差一定要出，公差有的很爽、有的很累，不過事情忙完可以投飲料。

然後最後一週會要游泳，長度是 50 公尺，深度約兩米，很容易就過去的。這部分就會游的就很爽，整個禮拜都爽爽過，不會的話，就盡量學，班長會很有耐心的教你。

手機的部分應該是大家最關心的，新訓是會把大家手機鎖在手機櫃，定點定時開放使用，大家只要表現好，基本上就很多時間能用手機， 手機用到都不怕兵變 。然後就切記不可以使用相機功能、開分享、定位。

打靶的話，聽說是要走一小時到靶場，但是我們那天下雨，所以搭軍卡上去靶場，沒走到真的有點可惜。然後我們在上面住了兩晚，晚上真的不要亂跑，很可怕、很不乾淨。
然後，打靶部分，基本上助教都會很仔細的教導，切記！不要白目、該做的動作要正確，不要嬉鬧，那就不會有太大問題了！

回家的話，外面有超多計程車，在營區門口就會看到一堆司機在拉人，我都是找人一起搭、分攤會很便宜。貴的話是貴在每週都搭高鐵回家，超燒錢…

下部隊

新訓待 42 天就會下部隊了，基本上部隊比較自由、伙食也好很多。

在下部隊前會大抽，我是 66 旅 部槍兵，基本上下部隊，兵種只是抽形式的，什麼槍都會去碰到、都要學，就盡力學、盡力記。那因為剛好我們營要下基地，我們不會跟到，所以基本上都是在幫忙下部隊的準備，掃地、檢整、打飯…等等。

我們剛好有遇到聯翔操演，基本上就是整晚不能開燈，然後小武裝，在妳守備的點蹲一整晚，記得要帶防蚊液！

還有遇到一次的夜行軍，基本上就跟著部隊一起走，然後走五十分鐘休息十分鐘，走 15 公里，整個又黑又無聊，但我覺得是個滿有趣的回憶。

下部隊後，時間過得更快，進去後，比新訓還好適應，會發現真的沒什麼，時間過很快。

總結

說真的，當初我進去的時候，怕到整個快哭了，但其實就適應後，真的就還好。

就造表操課，不去數日子的話，時間真的過很快！

然後在裡面就不要不長眼就好，是或不是、絕對服從，基本上就不會被刁難。

我一個弱弱的宅宅，平常也沒在運動、入伍前伏地挺身做五下就不行了，也都撐過來了，所以你們一定也可以！

報告班長，二兵___於海軍陸戰隊軍事訓練常備役 34 梯，2017/06/28 入伍 ，2017/10/11 退伍。

一日陸戰隊，終身陸戰隊！

永遠忠誠！

【原創小說】料理偵探

本作由 EricWu 製作，以創用CC 姓名標示-非商業性-禁止改作 3.0 台灣 授權條款釋出。

第一章 鑰匙

在這暮色籠罩的小鎮上，小鎮中間有座橋，這座橋的橋下沒有波瀾壯闊的浪花，僅有平平淡淡的河水流動著。

然而，在橋的彼端，有一個不起眼的偵探事務所，這個故事就從這說起。

事務所看起來很不起眼，外面是個普通的木門，門把上掛著一個小小的吊牌，吊牌上刻有一隻兔子的臉，兔子下面刻有一段字『歡迎光臨，小瓜事務所』。進入事務所，窗前有一組簡單的桌椅，桌上擺了很多雜亂的文件，看似是正在處理的案件、證據和各種線索，上面還有份不知道哪時候的報紙，桌角還放了一個聽說可以防小人，但是根本防不了的仙人掌。桌前則擺設了一個樸素的沙發，這沙發不外乎是給客人坐的。小瓜事務所就如同大家印象中的事務所那樣擺設，真的要說有什麼不一樣的話，大概就是事務所的主人和其他事務所有點不一樣。

偵探事務所裡面的負責人是為女性，名叫小瓜，年約二十幾，身材非常嬌小，一頭烏黑亮麗的長髮、一雙儘管臥蟬快消失但還是很迷人的眼睛、還有甜死人的笑容，不說還真的不會知道他是一個偵探。有時候都會讓人懷疑，平常來給她任務的人，究竟是因為長相慕名而來，還是真的有事相求才來。

然而小瓜平常的工作是協助調查鎮上的案子，他解決的案子已經多到不計其數，因此鎮上的人都稱她為『名偵探小瓜』。而這個名偵探小瓜養了一隻小小隻看似很可愛的兔子，這隻兔子確切的名字沒有人知道，只知道這隻兔子有強大的嗅覺，還有強而有力的嘴巴，能夠叼起很多意想不到的東西。

在偵探事務所對面有間小小間的餐廳，裡面住了一個外表看起來跟小瓜差不多年紀，但是心智年齡跟三十歲大叔差不多的男子，他一頭短髮，帶著一副文青的黑框眼鏡，身上有股淡淡的食物味道。他是這間店的老闆兼主廚，擅長料理，但是其實他更擅長的技能是密碼學，至於密碼學是什麼？他是數學和電腦科學的分支，他的原理涵蓋大量資訊理論，是個有趣的學問。雖然也不知道為什麼要介紹這，但搞不好下一集可以用上這個隱藏屬性也說不定，所以就先介紹一下了。

而這個名偵探小瓜最喜歡在晚餐時間帶著他的兔子，還有那些看似簡單、但不易解開的謎題來找老闆。就這樣這餐廳的主廚每晚都在和小瓜討論各種離奇的案子，都快要變成小瓜的御廚。

喔對，忘了介紹，這間餐廳的老闆兼主廚就是我，我是透尼。我這間餐廳開了快五年有了，雖然內外裝潢不是很起眼，甚至有點斑駁脫落，不過還是不影響我的生意。而我也很喜歡這份工作、喜歡這間餐廳、喜歡晚上討論的時間，還有喜歡小瓜。

你當真以為這故事會朝向什麼愛情故事發展，或是有什麼超展開嗎？當然不會，至少現在不會。

晚上的時候，小瓜總是會帶著各種各式各樣奇怪的案子來和我透過釐清思緒。我也不知道這鎮上為什麼有這麼多光怪離奇的案子，反正隔壁棚的江戶川先生，每集也都會有遇害者，聽說光是原作就死了 453 名。更別說另一棚的金少年了。所以這鎮上偶爾幾個遇害，應該也合乎常理，不然小瓜可能就真的只能在橋下當說書人討口飯吃了。

…

今天又是滿月。

正當我收拾餐廳準備打洋，邊擦桌子邊欣賞滿月的時候，就聽到門外的吵鬧聲，我大開大門，映入眼簾的不是月光，而是一堆鄰居在橋旁邊的一棟別墅外圍觀，我卸下廚師服，走了過去，只見擔架上一具遺體。

我端倪了一下擔架上的遺體，這具遺體帶了一頂毛帽，帽緣有些蒼白的白髮、身穿一件白色大衣、戴著口罩、穿著黑色長褲、手背看起來有些蒼老。

『應該是這棟別墅的主人…』恩，這是小瓜的聲音。我探出頭，朝著聲音的方向看。

我向小瓜揮了揮手，小瓜看了我一眼。

『這個人怎麼在哪看過啊？』

『好像是前面那戶人家的主人啊』

正當鄰居們議論紛紛著，小瓜飼養的兔子，突然跳了上擔架，往屍體聞了聞。

『兔兔！你不可以這樣』小瓜斥責了兔子。

只見兔子從屍體下面，叼出一把鑰匙，鑰匙上有點班黃、並且帶有一些血跡。看起來是個有歷史的鑰匙，而且不像是這個時代的鑰匙。

『這是什麼…』小瓜面有難色地看著這把鑰匙。而我很興奮地看著小瓜，不要問我為什麼興奮，你可能會以為是因為他要來找我討論這個離奇的案子了，所以我很興奮，但是絕對不是這樣，單純只是小瓜認真的表情太可愛了，所以我很興奮。

就這樣擔架隨著醫護人員扛起，只見救護車打開門，擔架送了進去，而在救護車後面跟著一群應該是他的子女的人。

第二章 造橋家族

這個小鎮的中心，有戶有錢人家，這家專門造橋鋪路，他們造的橋不僅工法流利、外觀出眾、速度又快、價格又便宜，因此很受歡迎。這個國家，大約有百分之八十七的橋都是這戶人家造的。然而，這戶人家女主人很早就過世了，而主人有三個女兒、兩個兒子。

恩沒錯，故事總是這麼剛好。這具屍體正是這家的主人覓雁先生。想當然，這麼有趣的案子，小瓜當然不會放過。

『啊啊啊啊，我明天還要早起還要營業誒！』我很無奈地被小瓜抓去陪她找這戶人家調查。

『又不會佔用你多少時間。』小瓜不屑的看了我一眼。

『抱歉！打擾了！』小瓜很禮貌的問了聲候。

『請進。』出來迎接我們的是一個女生，身材非常嬌小、綁著一頭流利的馬尾、臉上帶了點憂鬱、有著深邃的鎖骨、然後身材有點肉肉的。

『哎呀，小心點，撞倒我很麻煩的。』只見小瓜的兔子從小瓜手上鬆手後，開始活蹦亂跳，芷安才這樣警告小瓜。

『抱歉』小瓜連忙將兔子抱起來。

…

『我是大姐，我叫雅涵。』

『我是二姐，我叫于庭。』二姐不懷好意的笑著

『我是最小的，我叫家妤。』這女生看起來非常的害羞。

『我叫易南，我是大哥。』大哥看起來年約三十，一頭亂髮，係著綠色的領帶，給人豪放親切的感覺。

此時，坐在一旁的芷安開口了。

『我是二哥，我叫做圍城。然後排邊這位是我的太太芷安。』

『人，是我殺的。』坐在二哥旁邊的氣質女子，衣服有不少污漬，也就是剛剛來幫我們開門的芷安開口說話了。

『誒？』小瓜一臉問號的喊了一聲。

『因為父親長期欺負我，我實在受不了…』芷安淡定地講這。

『對啦對啦，人是他殺的！』大哥慌張地說了。

此時，芷安端出了一件紅色的領帶。

『看吧，這就是我殺人的痕跡。』

就在這個時候，兔子跑過去聞了聞上衣，舔了好幾口。

『誒！兔兔你壞壞！』小瓜慌張地把兔子抱了過來。

我覺得很奇怪，所以我也靠過去聞了聞，這領帶上貌似有紅酒的味道。

『誒！怎麼連你也靠過去聞… 你是兔子嗎！』小瓜整個白眼翻到後面去？

『人就是他殺的啦』此時屋內一陣騷動，三姐妹有點騷動，紛紛一起茅頭指向芷安。

『是的，因為父親長年病痛，所以我們請了人來照顧他。父親一直以來都很難伺候，這點我們很清楚，不過我們真的沒想到會照顧她照顧到痛下殺機…』大哥開口說話

『我們相信他一定不是故意的…』二哥帶有點哽咽的說著。

『今天…恩…就先到這吧』小瓜抱著兔子，覺得整件案件充滿疑點。

正當我也陷入一陣沈思的時候，小瓜過來點了我一下。

『誒！明天晚餐我要過去你那吃喔！』小瓜看著我說。

『你今天想吃什麼？』雖然我知道他想吃什麼，但還是禮貌的問一下。

『隨便啦』小瓜一臉期待的眼神看我。

『阿順便要幫兔兔準備食物喔，他今天可是陪著我們忙了一天，所以要加倍食物招待喔』他一臉更期待的看著我了。

比起晚餐，我更好奇這件案子的兇手啊，想到這我就根本無心思考晚餐。

第三章 推理？先吃飽！

『今天生意不錯嘛』我邊準備晚餐，邊看今天的收入報表。

『我來了。』小瓜打開門，冷冷地說了一聲。

只見滿桌的法式料理，前菜之前，我先提供了「純手工法國麵包佐特調抹醬」，麵粉味很香，搭配上帶有自己特調的醬料，為了避免各位去對面開一家，所以姑且不聊這特調醬料了。

『那個…昨天的…』小瓜看了一下餐桌，先開口了。

『吃飽再來討論吧』我用手指壓住了她的嘴唇暗示他不要再說了。其實是因為我現在肚子超餓的，不想推理啊。

接著上的這道料理的，名字超長的，全名是「鑑賞級挪威煙燻鮭魚薯泥沙拉佐小黃瓜酸奶油」，不要問我為什麼要這樣命名，不覺得法式料理就是要這樣命名嗎？

這道料理配色看起來十足繽紛、吃起來也十分繽紛，底層先塗上少量的酸奶油、再塗上少許的薯泥，上面疊著三片的煙燻鮭魚，最後旁邊在放上小黃瓜作為點綴。平常不太喜歡吃煙燻鮭魚的人，常常是因為怕煙燻的味道很重、或是擔心鹹味，但是這樣搭配可以完全沒有這問題，很適合當開胃菜。

吃到第二道，小瓜有點不開心的開口問說『為什麼不是火鍋！』

是的，小瓜最喜歡的是火鍋，但是我卻沒煮，我看著他說：『你知道嗎？法式料理的哲學嗎？就是美食猶如戀愛，你必須和食物產生美好的關係！所以不單單只是重視食物好吃與否，更重要的是整體氛圍。因此在烹調的過程中，總是用盡心思將最美好的味覺、視覺感受投入料理，進而去創造出幸福的體驗。而我，也想用盡心思將最美好的味覺、視覺感受投在你身上，創造幸福給你。』

小瓜一臉驚恐地愣了三秒，然後冷靜下來看著我問『那昨天的日式料理呢？』

『不！這你就有所不知，日式料理的哲學你知道嗎？就是對每一種食材都要很認真的對待與相處，就像凝視愛人般，關注著食材的表情，抓住著食材的美味瞬間。凡事都要為另一半著想，也就是上菜的那一刻，不，從客人進入店裡的那一瞬間，廚師的愛意就萌生了，主廚必須去發自內心的體察客人的內心，從料理上食器、擺盤、每道餐點的間隔時間，每一個小細節都不能馬乎，這正是日式料理的哲學。』

我想小瓜大概什麼也沒聽進去吧，只見他默默地吃了起「鑑賞級挪威煙燻鮭魚薯泥沙拉佐小黃瓜酸奶油」的煙燻鮭魚。

『哎…又被無視了』我內心這樣想著。

接著上菜的是湯品『法式經典蘑菇湯』，這道料理有兩個碗，一個是湯碗，另一個則放滿蘑菇料，食用前要將湯碗裡的高湯慢慢倒入放滿蘑菇料的碗，之所以要這樣做，是為了保持清湯的原味。因為是清湯，因此味道相對清淡爽口，在主餐來以前先淡淡嘴裡的味道，簡直完美。

主餐時間終於來了，『法國頂級牛肋眼佐蘑菇醬汁』，肋眼的大小適中，油花分佈均勻，因此咬下去相當的有嚼勁，卻又不會太韌，搭配特調的蘑菇醬因此不會太膩。

就在享受主餐的時候，我消失了一陣子。那是因為，最後是甜點「舒芙蕾」。

在法式甜點中，舒芙蕾是一個最有挑戰的甜點。

舒芙蕾難就難在，他一定要現做現吃，所以只有『人等舒芙蕾，而沒有舒芙蕾等人』。相傳在法國餐廳，點了舒芙蕾，一定會被告知要等上四十五分左右，要等這麼久，並非舒芙蕾真的要製作這麼久，而是為了預留時間，以免廚師搞砸了舒芙蕾而必須重作一個，因為即使廚藝再高強的廚師，都不敢保證能百分之百的做出完美的舒芙蕾，製作過程常常些無法預料的事，舒芙蕾就這樣突然塌陷了。

就算廚師完成了完美的舒芙蕾，也必須在三分鐘內就能從廚房送到客人桌前，客人必須趁熱馬上品嘗，否則舒芙蕾遇冷就會開始塌陷。所以，法國人還有一種說法，如果剛好有一桌五個以上的客人，同時都點舒芙蕾，極有可能是來餐廳搗亂的。

因此，完美的舒芙蕾是需要天時地利人和的，也夾帶著很多運氣的成分，就像人生一樣，人生有很多令人意想不到的過程，常常就和舒芙蕾一樣脆弱。

『吃飽，可以推理了。』就在小瓜吃下最後一口舒芙蕾，我開口說了。

這時候我是不是應該點根雪茄、或是抽口煙，然後把煙往上吐，再長嘆一口氣，帥帥的下個結論。

但是想起剛認識小瓜的時候，他說過要是我吸菸就不跟我當朋友，所以我現在只能邊洗碗盤，邊跟他討論。

『我覺得這不單單只是一個怨氣累積後，單純的殺人事件』小瓜憑著她過人的觀察力和第六感得到的結論。

『所以你想要去調查？』我故意問他。

『不然勒？你真的很不解風情誒』小瓜瞪了我一眼。

…

話說上次芷安認罪後，就被羈押到看守所了。所以我一早就拉下餐廳鐵門，被小瓜抓去看守所找芷安。

只見芷安拖著疲憊的身軀出來見客。

『你可以簡單的，說一下當晚發生什麼事情嗎？』小瓜先開口問話。

『當天我正送晚餐去給父親吃，然後父親吃了一口，就翻桌說不好吃不吃了，於是我一氣之下，就拿領帶把父親給…』

『因為東西不好吃，就浪費！不可原諒！』身為廚師對於浪費食物這件事情一定要唸一下。

『那可以問一下你當天的菜單嗎？』

『當天晚上因為我很忙，所以簡單做了炒飯，你們問這麼多幹嘛…不是都說人是我殺的了？』

我冷笑了一聲。小瓜面有難色地看著芷安不悅地轉身離開。

第四章 真相

我們又再次去拜訪了這戶人家。

『恩，今天我們想來實地調查一下。』

『那我就為你們做介紹吧』大哥這樣說。

『首先這裡是大廳。』全家人都坐在大廳，看起來坐立難安的。

『左邊這裡是餐廳，平常大家是都在這吃飯的，不過父親通常都在樓上自己的房間吃飯。』

『右邊這間沒有門的是書房，平常我都在這裡習讀造橋的知識。』

『再來我帶你們去二樓。』

我們就跟著大哥的腳步上了二樓

『再來這裡是左邊這間有拉門的是主臥室，通常老闆都睡這…』講這句話的時候，大哥的表情顯得特別哀傷。

『再過去依序是我的、二哥、大姐、二姐、小妹的房間。』

『最後一個房間我不方便說。』

依序介紹完後，我們回到了大廳。

『好的，那我再來想要跟你們單獨問話。』小瓜對著大家說。

『為什麼？』二姐開口問，其他人也對於單獨談話這件事有異議，議論紛紛的。

『人類記憶很曖昧的，就像是你對大碗咖哩飯的記憶也不同，有些人覺得小小幾口就是大碗，有些人則覺得要滿滿一鍋才是大碗。』我這樣說。

…

首先是大哥的單獨問話。

『我當天正在參加商人協會的聚餐，所以發生什麼事情…其實我也有點混亂。我只知道我趕回去的時候，就看到父親拿著橘色領帶，然後芷安錯愕的看著父親。』

『好的，看來這沒什麼好問的，下一位吧』小瓜說。

接著是，二哥的單獨問話。

『那天我正在書房看書，突然聽到父親的慘叫，我就趕快上樓看，就看到父親拿著橘色領帶，然後芷安錯愕的看著父親。』二哥說。

『我正在庭園澆水，我聽到慘叫聲，我就趕快跑去二樓，一上樓就看到父親拿著橘色領帶，然後芷安錯愕的看著父親。』第三位是大姐。

『當時我身體不適，在房間休息，聽到慘叫聲後，我就從隔壁跑過來，一開門就看到父親拿著橘色領帶，然後芷安錯愕的看著父親。』接著是二姐。

『我…當時在房間插花，突然間，聽到慘叫聲，我就走了過來，就看到父親拿著橘色領帶，然後芷安錯愕的看著父親。』

於是花了一個多小時，終於單獨問話完，我們就結束了行程回到我的餐廳。

…

『頂罪』小瓜點著頭說。

『你為什麼這樣覺得？』雖然我內心也這樣認為，但我想聽聽其他人的看所，所以我就沒有因此附和。還有，此刻的我真的覺得，吐一口煙、長嘆一口氣，再喝杯咖啡真的比較帥，不過可惜的，現在的我只能擦拭著碗盤。

『第一點、大家證詞居然都一樣。』小瓜說。

『而且這領帶應該是紅色吧，為什麼大家都說是橘色？』我皺起眉頭回答。

『第二點、當天晚餐是吃炒飯，為什麼衣領上有紅酒？』

『人家不能炒飯配紅酒嗎？』我笑著看小瓜說。

『不過問題是，地板上沒有紅酒的痕跡，這太不自然了！』我表情一皺，看著小瓜。

『對～芷安身上也沒有紅酒痕跡，那領帶的紅酒是哪來的？』

『還有一點、你不覺得突然出來認罪這件事情，感覺很不自然？』

『我也這麼認為』看來我們推理的方向有共識了。

『是因為被威脅頂罪嗎？』我這樣說。

『誰知道？對了！芷安小姐是不是有懷孕？』

『感覺有喔』我笑笑地結束了今晚的話題，因為等下我有更重要的事情要做。

小瓜離開後，我上了樓撥通電話，對方是商人協會的朋友，有些事情我想確認。

…

早上的時候，有個壯碩的男子來找我，給了我張照片，我看了後冷笑一下，打了電話叫小瓜準備準備我們要出發了。

就這樣我們又再次來到這別墅。

全家人又坐在大廳，這次氣氛依舊非常凝重。

『芷安的罪行依以往經驗會被判多久？』二哥激動的說。

『可能可以減輕罪刑，甚至是免罪，因為真正的犯人還沒找到。』我很有自信的這樣說。

『大哥，你說你那天是去聚餐吧？你還記得你的領帶是什麼顏色嗎？』

『綠色啊』

『很好，沒有說謊』我這樣說。

『你現在是在懷疑我？』此時的大哥看得出來很憤怒。

『這是你當天的活動照片。』我從口袋拿出一張照片。

『你的領帶可是紅色的喔，喔不你們說那是橘色。難道你回來看父親遺體還要換領帶？』

『還有就是，芷安衣服上並沒有紅酒的痕跡，為什麼這條領帶會有？既然翻桌把食物灑到衣服上，照理講兩樣物品沾到的污漬應該是一樣的吧？這紅酒是你去吃飯的時候犯蠢沾到的吧？』

這時候，小瓜拿起了一個密封袋，裡面裝有一把鑰匙，原來是上次在大衣上的鑰匙。

『你要的是這個吧？』

『你怎麼會有這個！』大哥慌張地看著小瓜。

『這是你想要的吧，我們調查過了，這是你們的傳家之寶，聽說在二樓有個房間，放滿造船技術的秘密，而這把鑰匙正是這房間的鑰匙，對吧？』

『夠了！別再說了！』二姐很激動的說。

…

『是這樣的…那天晚上我提早離開，我在父親房間跟他討論鑰匙的問題，可是家父覺得我還很弱，不能繼承給我…。就這樣我們起了衝突，然後我把身上的領帶拿來做了傻事情….勒在父親的頸上』大哥跪在地上邊哭邊講。

大哥開始敘述當時的過程…

然後父親很痛的慘叫，大家紛紛過來。

『你怎麼做了這種傻事…』二哥說。

『為什麼….』二姐哽咽的說。

『就由我來承擔吧！』芷安把大哥手上的領帶搶了過來。

『就說是我常常被父親欺負，所以心懷恨意，殺了他的。』芷安繼續說。

『對對對，這樣大哥才能好好專心拼事業，反正我們找個好的律師，很快就能讓芷安出來了！』大姐激動的說。

『那…那我們要先串供，就說我們上樓，就看到父親拿著紅色領帶，然後芷安錯愕的看著父親。』二哥接著說。

『對對對，就看到父親拿著紅色領帶，然後芷安錯愕的看著父親。』二姐跟著起鬨。

『不可以說是紅色，要說是橘色，因為父親平常都這樣說。大哥你趕快去換衣服吧，待會警察要來了，到時候被發現就不好了。』

『好！那我們就改說就看到父親拿著橘色領帶，然後芷安錯愕的看著父親。大家有記得嗎？』大哥這樣說。

…

大哥敘述完後，抱頭痛哭。

『這樣哭很難看啊…只要在同一個屋簷一下一起變老就是一家人，拿自己家人來頂罪也好好反省吧你…！』

『我…』大哥整個哭得更大聲。

第五章 後續

一個芷安後的某個晚上，我正在和小瓜討論新的案子。

一個女子來到我的餐廳，身材非常嬌小、綁著一頭流利的馬尾、她有著深邃的鎖骨、然後身材變得更肉了，還有臉上不再憂鬱了，對就是那時候的那位芷安。

『我來跟你們說謝謝的』

『為什麼一開始不說真話。』我邊擦桌子邊說。

只見芷安看了我一眼。

『因為我覺得大哥應該要繼承家業，所以我是最適合的人選，應該把最扛下來。』

『你懷孕了吧。』我放好抹布後，邊倒茶邊說。

芷安靜靜的不回話。

『事實上，你故意把餐點弄在身上，還有把領帶從紅色說成橘色，都是故意引誘我們找出真相』我喝了口茶繼續說。

『我這樣做有什麼意義？』芷安說。

『代位繼承。』我看著茶杯說。

『你在說什麼，一點證據都沒有。』只見芷安冷笑了一聲。

『是的，沒有證據喔。』我也冷冷的回了一聲。

人生百態、家庭百態，每把鑰匙對應了一個空間、一個空間也對應了一個人生。打開鑰匙進入這空間，猶如人進入了其他人的人生中，也意味著正在改變那個人的人生。而人與人之間總為了貪婪而再次改變那個人的人生。

而在我旁邊的這位女生，是這個鎮上最天才的偵探，名偵探小瓜，她如同醫生，擔負了病人的生命一樣，也背負著每個光怪離奇的案子，努力為眾多的案子找出真相。

而我是個料理偵探，看似靜靜的做著這些料理，事實上默默的關注著這些案子。

-終-

在 Weebly 上使用 Gandi 的域名

由於朋友最近正在研究 Weebly ，提到了『自定義域名』功能，所以我也來研究了一下，雖然我覺得我應該不會再用到，不過還是紀錄下來。

自定義域名 功能允許您設置您的 Weebly 網站綁上自己的域名。首先你必須先尋找、購買新的域名，並且在 weebly 設定好。

  • 登錄 Gandi，選擇我的帳戶 。

  • 點選你要指向的域名，並單擊進去管理頁面。

  • 在最底下，會有個區域檔，選擇編輯區域。

  • 你應該會看到一個警告訊息，這是 Gandi 為確保你的網域正常運作所做的機制。你只需要點選右邊的 建立新版本

  • 接著找一下 Host 欄位是 @的 ，點選他右邊的鉛筆。

  • 在 值 欄位中，填寫上 weebly 提供的 IP 地址之一。Weebly 提供的的IP地址有 199.34.228.55，199.34.228.56，199.34.228.57，199.34.228.58，199.34.228.59，199.34.228.100。

上述選一個就好（如果你有使用到 SSL，那麼你就需要使用特定的IP地址）。如果你覺得這步驟很繁雜的話，你就直接填入 199.34.228.59。 填好後，點擊送出按鈕。

  • 接下來，向下滾動頁面，以確保www的CNAME記錄指向@。如果沒有，記得要修改它。

這樣就完成了！

接著 Gandi 的可能需要長達 24 小時的時間來更新他們的記錄（通常很快，幾分鐘就會更新完畢了）。

在 Raspberry pi 3 上面使用 waveshare 3.5 吋 TFT 螢幕

在 Raspberry pi 3 可以直接透過 HDMI 輸出畫面到螢幕上，不過這次參與 Crego 上面開的坑，目標是做出小型遊戲機，因此我們要使用 retropie 這個 O.S.。必較麻煩的是，要額外將 3.5 吋 TFT 螢幕接到 Raspberry pi 的 GPIO 腳位。

我一開始撞牆滿多次，不過其實滿簡單的，所以就紀錄一下實作方法。

  • 首先預備工作，要先下載三個檔案：

    • 下載 TFT 螢幕驅動程式
    • 下載 fbcp 套件包
    • 為了讓驅動程式可以在 retropie 上運行 下載驅動程式的解決方法

  • 將上面三個檔案都丟到 /home/pi 路徑下。

  • 要進行 SPI 設定

    • 在 terminal 上輸入 sudo raspi-config
    • 進入 advances settings ，開啟 spi

  • 在 terminal 上執行 tar xvf LCD-show.tar ，來解壓縮檔案

  • 編譯 fbcp

    • 執行 cd fbcp 進入 fbcp 資料夾
    • 執行 sudo mkdir build 創建資料夾
    • 執行 cd ./build 進入 build 資料夾
    • 執行 sudo cmake ..
    • 執行 sudo make

  • 到這步已經差不多了，接著只要執行 sudo bash /home/pi/driver_workaround.sh，此時會看到螢幕由白變黑。

  • 最後，執行 sudo /home/pi/rpi-fbcp/build/fbcp &，就會看到畫面了。

備註：這次我是採用 3.5 吋 LCD 螢幕, 所以無法確定其他尺寸的 GPIO LCD 螢幕，是否可以運作。
