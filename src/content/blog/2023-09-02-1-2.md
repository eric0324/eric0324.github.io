要架設網站實在不是很困難的事情，相對應的架站工具也很多，像是 Wix 跟 Weebly 也是不錯的選擇。那在這麼多選擇之中，我們為什麼要選擇使用 WordPress 呢？

WordPress 誕生於西元 2003 年，是一個開放原始碼軟體，正因為 WordPress 本身是自由開源軟體，所以大家可以免費下載與安裝使用。並且在全世界，都有開發者開發外掛、也有設計師貢獻佈景主題，或者是很多志工在貢獻本地化的翻譯，成了一個著蓬勃發展的生態圈。

順帶一提，WordPress 在拼寫的時候， W 和 P 務必要大寫，這是一種對品牌的尊重。就像是應該寫 LINE ，而不是 line ；應該寫 GitHub 而不是 github。

<!--more-->

大家對 WordPress 最早的印象，可能是拿來做部落格。隨著外掛生態系蓬勃發展，能做的功能也越來越豐富了！很多人不只拿來製作個人**部落格**，也會用來架設**形象網站**、**電子商務網站**、**甚至線上學習平台**。

在以前，架設網站可能會是個需要點技術門檻的事情。隨著 WordPress 這個開源軟體，這十幾年來不斷地進化，經歷多次更新改版之後，至今已經到了第六代的 WordPress，讓網站架設已經不在像以前這麼的困難。

另外，截至目前為止，地球上接近**四成的網站**，都是用 WordPress 所架設的！也就意味著有豐富的生態系和社群。也就是說，今天你可以很容易的在網路取得豐富的資源！

對於新手來說，如果製作上有些想法或是遇到一些困難的話，都可以在 Facebook 社群或是 Slack ，透過這些線上資源 (在之後的章節，我也會和大家分享如何取得這些資源) 找到人來討論。不僅只有網路上有豐富的資源，線下也有各地區的定期小聚，有任何想法的話也可以直接來線下做交流。另外，每年在世界各地也有大型年會 WordCamp ，可以和許多使用 WordPress 的人做交流，所以你不孤單。

正因為 WordPress 有這強大的社群，所以很多開發者願意替 WordPress 開發好用的外掛或是設計好的主題。如此一來，你不用自己去撰寫程式碼，就可以做電子商務網站，你不用懂程式架構，你就可以設定 SEO。

我們可以很輕易的直接從 WordPress 後台搜尋與挑選外掛來安裝，看到喜歡的可以直接點擊安裝跟啟用（而且很多都是免費的！），除非你要使用進階功能，才會需要額外付費購買。

沒錯，我們只需要透過 WordPress 的外掛，就可以輕鬆製作出好看又好用的網站，你不需要太多的設計與程式的基礎。
