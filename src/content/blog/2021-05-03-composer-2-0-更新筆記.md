[  
Composer](https://getcomposer.org/) 2.0 已經 release 很久了，主要是整體速度提昇了很多(提昇了 50% 以上的速度，實測後相當有感)，所以手下的電腦都陸續更新到 2.0 了，不過我每次都會忘記更新的指令和方法，所以這邊紀錄一下

<!--more-->

關於 Composer 2.0 釋出的相關資訊，請參考：

- [Composer 2.0 is now available!](https://blog.packagist.com/composer-2-0-is-now-available/)
- [Composer 2.0 is now released with performance improvements](https://laravel-news.com/composer-2)

## Composer 2.0 升級步驟

```
composer self-update # 升級 1.x
composer self-update --1 # 若不想升級到 2.0，強制只升級 1.x 版
composer self-update --2 # 升級到 2.0版
```
